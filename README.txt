This suite of tools tests an IPP Client or IPP Printer's compliance and level of compliance with the recommendations in the IPP Implementor's Guide v2:

http://ftp.pwg.org/pub/pwg/candidates/cs-ippig20-20150821-5100.19.pdf

To test a Client, run "test-ig-client.sh" and follow the instructions output by that script.

To test a Printer, run "test-ig-printer.sh" and follow the usage instructions output by that script.
